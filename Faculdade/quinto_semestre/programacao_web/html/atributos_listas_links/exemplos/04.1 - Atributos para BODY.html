Como foi dito no slide anterior, é na tag <body> que colocamos todo conteúdo do site, e nela podemos definir cores para os texto, links, plano de fundo ou imagens de fundo.

Atributos

text: define a cor do texto da página. O padão é preto.
link: define a cor dos links do site. Por padrão a cor é azul.
vlink: cor do link visitado do documento. Padrão roxo.
bgcolor: muda a cor do Plano de Fundo. A cor padrão é branco.
background: coloca uma imagem ao invés da imagem.

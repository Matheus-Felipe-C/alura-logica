FSA Sites



Desenvolvimento e Criação de Sites

Desenvolvemos soluções personalizadas compatíveis com a maioria dos dispositivos (smartphones, tablets, notebooks) baseado nos Padrões WEB, tornando fácil a indexação nos motores de busca. Ou seja, você é encontrado e a navegação é garantida!

Tipos de sites que desenvolvemos
  • Site Estático: Seria um site com poucas modificações futuras, não existe um painel de administração para gerenciá-lo.

  • Site Administrável: É um site dinâmico, que permite a atualização através de um painel de administração protegido por senha.

  • E-commerce - Loja virtual: Site para vender na internet, com gerenciador de produtos, clientes, além de cálculo de frete automático.

  • Portais: Site de conteúdo, mais complexo e completo que o Dinâmico. Feito para quem quer prover informações na internet.

  • Sistemas Personalizados: Uso frequentemenete empresarial, para necessidades específicas, sob medida para seu negócio.

  • Mobile Site: Adaptado para dispositivos móveis, como smartphones, tablets, com uma interface fácil e rápida para ser navegada.
